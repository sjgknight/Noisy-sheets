Noisy sheets is an add-on that works in google spreadsheets, which allows users to: add noise to a dataset; share the unique datasets with a student cohort; and calculate function results for the students' versions of the data.

Noisy sheets add-on has two options in the menu: 1) "Add noise", which, as its name states, adds noise to a dataset. A number of options are provided to fit your new dataset; and 2) "Answers", which creates a new spreadsheet file named "MasterAnswers". This file contains the correct answers for calculations on the datasets provided to students. The answers in this master sheet can then be compared to student quiz responses.

Before using the "Noisy sheets" add-on, please bear in mind the following suggestions:

Add Noise

  • The original spreadsheet should be in a different folder rather than the main “My drive” folder.
  • The original spreadsheet should have 3 sheets in total:
    • The dataset
    • A sheet with the students’ information in the following format: name, last name, and email.
    • A sheet with the questions to be asked to students in following format: questions and answers.

Answers

  • Before generating the answers, you should share the spreadsheets with students using the “Share” button in the “Add Noise” option.
